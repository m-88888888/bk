# bk

bk bookmarks your directory.

## Install

```sh
$ make build
$ make install
```

## Requirements
- peco

## Usage

- bookmark
```sh
$ bk save
```

- show your bookmarks
```sh
$ bk show
```

- delete your bookmark
```sh
$ bk delete
```

- jump your bookmarked directory
```sh
$ jp
```

## Uninstall

```sh
$ make clean
```
